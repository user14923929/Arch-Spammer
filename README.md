
# Arch Spammer

A simple PowerShell script that repeatedly displays the message "I use Arch btw".

## Overview

This project contains a single script that runs an infinite loop, outputting the popular tech community phrase.

## Files

- `main.ps1` - The main PowerShell script

## Usage

```powershell
.\main.ps1
```

The script will continuously print "I use Arch btw" to the console. Press `Ctrl+C` to stop execution.

## Requirements

- PowerShell
